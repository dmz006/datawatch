# Plan: Patch 0.6.1 — Backlog Bugs and Docs Review

## Source
Backlog review of `#docs`, `#bugs`, `#updates` items post-v0.6.0.

## Items Being Fixed

### Bugs
1. **channel_enabled not default true** — `DefaultConfig()` in config.go does not set `ChannelEnabled: true`. Since channel is now self-contained (Node.js embedded), it should be on by default. Fix: add `ChannelEnabled: true` to `SessionConfig` in `DefaultConfig()`.

2. **Input field auto-focuses on mobile causing keyboard popup** — `inputEl.focus()` in `renderSessionDetail()` (app.js:608) fires unconditionally. Fix: skip focus on touch devices using `navigator.maxTouchPoints > 0 || window.matchMedia('(pointer:coarse)').matches`.

3. **Cannot send to tmux when in channel mode (non-waiting)** — When `sessionMode === 'channel'` and not waiting_input, only "ch" send button shown; no way to send to tmux. Fix: add a small toggle (▶ tmux) secondary button alongside "ch" button, storing preference in `state.sendMode[sessionId]` (defaults to 'channel'). When toggled to tmux, `sendSessionInput()` is called instead.

4. **Web UI update button misleads** — `runUpdate()` in app.js sends the `update` command which only CHECKS for updates (calls `handleUpdateCheck()` in router), then shows "Updating…" toast. Actual install requires `datawatch update` on CLI. Fix: (a) change the button label/flow to be honest about what it does, (b) add a `POST /api/update` endpoint that runs `installUpdate()` in a goroutine and streams progress, (c) add self-restart via `syscall.Exec` after successful install.

5. **Android Chrome "notification permission denied"** — Generic error toast gives no guidance. Fix: detect denied vs unsupported and show actionable message for Android (check site settings > Notifications).

6. **Uninstalled LLM backends appear selectable** — New session form shows `(not installed)` label but backend is still selectable. Fix: mark the option `disabled` so it cannot be selected.

### Docs
7. **opencode-acp: document what commands work over ACP vs tmux** — Add table to `docs/llm-backends.md` opencode-acp section matching the claude-channel.md style.

8. **CHANGELOG missing v0.5.13–v0.5.20** — Reconstruct entries from git log and add to CHANGELOG.md between v0.5.12 and v0.6.0.

### Updates
9. **Restart after update** — After `installUpdate()` succeeds, call `syscall.Exec(selfPath, os.Args, os.Environ())` to restart the daemon in-place with the new binary. Add to both the CLI `runUpdateCmd` and the new `/api/update` endpoint.
